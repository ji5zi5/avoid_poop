# Boss Pattern Expansion Design

**Date:** 2026-04-12

## Goal
보스 구간의 단조로움을 줄이기 위해 대왕똥 기반의 대형 압박 패턴과 함정형 패턴을 추가한다. 일반 모드는 1~2번 보면 읽히는 수준, 하드 모드는 외워야 피하는 패턴 일부를 허용하되 불합리한 조합은 금지한다.

## Current Problem
- 보스 패턴이 동일한 레인 낙하 반복 위주라 체감 변화가 작다.
- 대형 엔티티가 없어 보스의 위압감이 부족하다.
- 패턴 간 성격 차이가 약해 일반 구간보다 약간 더 빡센 정도로만 느껴진다.

## Design Summary
- 보스 패턴 엔진을 `랜덤 패턴 묶음` 방식으로 바꾼다.
- `대왕똥` 타입을 추가해 화면 절반을 덮는 압박 패턴을 만든다.
- 보스 시작 시 준비된 패턴 풀에서 3~5개를 선택해 순서대로 실행한다.
- 일반 모드는 읽히는 패턴 위주, 하드 모드는 함정형 패턴 전부를 사용한다.
- 패턴별로 전조, 본체 낙하, 후속 함정, 종료 조건을 분리한다.

## Pattern Pool
### Core Pressure
1. 절반 막기: 화면 좌/우 절반을 빠르게 번갈아 덮는 대왕똥
2. 양문 닫기: 좌우에서 큰 똥이 들어와 가운데만 남김
3. 중앙 압착: 중앙을 덮고 양끝만 열림
4. 외곽 압착: 양끝을 덮고 가운데만 열림
5. 연속 덮기: 같은 방향 2연타 후 반대 방향 전환

### Lane Variants
6. 통로 흔들기: 안전지대가 좌우로 크게 이동
7. 지그재그 통로: 안전지대 위치가 단계적으로 튐
8. 스위치 압박: 막힘 방향이 빠르게 스위치됨
9. 교차 낙하: 큰 똥과 작은 똥이 시간차로 교차

### Trap Variants
10. 가짜 안전지대: 비어 보이는 쪽에 후속 낙하가 이어짐
11. 잔류 함정: 큰 똥이 지나간 자리 일부가 짧게 위험 상태 유지
12. 페이크 전조: 경고와 실제 낙하 방향이 어긋남
13. 지연 폭주: 전조 후 짧은 정적 뒤 고속 낙하
14. 막타 함정: 패턴 종료 직후 작은 후속 낙하 추가

## Difficulty Policy
### Normal
- 14종 중 읽기 쉬운 8~10종 사용
- 전조 시간 길게
- 안전지대 폭 넓게
- 함정 패턴 연결 길이 짧게

### Hard
- 14종 전부 사용
- 전조 시간 짧게
- 안전지대 폭 좁게
- 함정형 패턴 연속 사용 허용

## Fairness Rules
- 플레이어 히트박스 기준 최소 안전 폭 보장
- 전조 후 최소 반응 시간 보장
- 절반 막기와 후속 잡몹 조합 시 완전 봉쇄 금지
- 잔류 함정과 즉시 반전 패턴을 같은 프레임에 겹치지 않음
- 하드여도 회피 불가능 상태는 생성하지 않음

## Data/Code Changes
- `GameState`에 보스 패턴 큐, 현재 패턴, 패턴 단계 타이머를 추가한다.
- `bossPatterns.ts`를 단일 함수에서 패턴 정의 집합 + 시퀀서 구조로 재편한다.
- `Hazard`에 타입 또는 크기 기반 분기를 추가해 대왕똥 렌더/충돌을 처리한다.
- 렌더러는 일반 똥, 대형 똥, 보스 대왕똥을 구분해서 그린다.
- 전조 UI는 기존 배너를 재사용하되 패턴별 짧은 경고 표시를 추가한다.

## Testing
- 패턴 선택 중복 제한 검증
- 일반/하드 모드 패턴 풀 차이 검증
- 절반 막기 안전지대 보장 검증
- 페이크/함정 패턴도 최소 반응 시간 보장 검증
- 대왕똥 충돌 크기와 렌더 크기 일치 검증
